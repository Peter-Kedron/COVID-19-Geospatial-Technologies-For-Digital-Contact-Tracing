Status of State Digital Contact Tracing Technologies

Our project team is tracking how states are using digital contact tracing technologies (DCTT) to respond to COVID-19. The map and table below contain a summary of which states are using which technologies in their response. We will be regularly updating and expanding this information as the project progresses. More information about individual DCTT is available on our Technologies Page or through the Project GitHub Repository and the Project Open Science Framework Page.
